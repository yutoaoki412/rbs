よくあるご質問

Q1. レッスンではどのような内容を行いますか？ +

RBS陸上教室では、「走る・跳ぶ・投げる」といった陸上競技の基本動作を中心に、遊び感覚も取り入れたトレーニングを通じて自然と運動能力を高めていきます。短距離を中心に、他の球技にも活きるトレーニングを楽しく体験しながら基礎を習得できるカリキュラムです。

また、非認知能力（挑戦する力・集中力・協調性など）を育むプログラムも導入しており、競技力だけでなく人間力の向上にも力を入れています。少人数制クラスで、一人ひとりの成長に寄り添った丁寧な指導を行っています。

Q2. 無料体験は可能ですか？ +

はい、無料体験レッスンを随時受け付けております。体験をご希望の方は、当教室のウェブサイトにてお申し込みください。

Q3. 他の陸上クラブとの違いは何ですか？ +

RBS陸上教室は、単なる技術指導だけでなく、子どもたちの非認知能力（自己肯定感、協調性、課題解決能力など）を育むことにも重点を置いています。また、個々の成長に合わせた指導を行い、子どもたち一人ひとりの可能性を最大限に引き出します。

Q4. 持ち物は何が必要ですか？ +

運動に適した服装、運動に適したシューズ、飲み物（給水用）をご持参ください。また、屋内施設を利用する場合は、室内履きが必要となることがありますので、事前にご確認ください。

Q5. 欠席した場合、振替はできますか？ +

はい、月に1回まで他のクラスへの振替が可能です。また、天候不良など教室側の都合でレッスンが中止となった場合は、別日に振替を行います。

Q6. 雨天時のレッスン開催についてはどうなりますか？ +

屋外でのレッスンの場合、雨天時は中止となることがあります。レッスンの開催可否は、開始1時間半前までに当教室のウェブサイトの「本日のレッスン開催状況」セクションにてお知らせいたします。

Q7. 休会は可能ですか？ +

はい、休会をご希望の場合は、休会される前月の20日までにご申告ください。休会は月単位で承っており、復帰月を事前にお知らせいただく必要があります。

Q8. 月謝の支払い方法は？ +

基本的には自動口座引き落としをお願いしております。一部の教室では、銀行振込など他の支払い方法にも対応しておりますので、詳細はお問い合わせください。

Q9. 何歳から参加できますか？ +

当教室では、年長からご参加いただけます。体験レッスンは、該当年度の4月1日時点で年長を迎えているお子様が対象となります。

Q10. 保護者の見学は可能ですか？ +

はい、レッスンの見学は可能です。お子様が緊張している場合など、保護者の方が一緒に参加いただくこともできますので、ご相談ください。
