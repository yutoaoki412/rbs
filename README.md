# RBS陸上教室 - Webサイト & 管理画面

## プロジェクト概要

RBS陸上教室の公式Webサイトと管理画面システムです。保護者向けの情報提供と教室運営の効率化を目的としています。

### 特徴

- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ完全対応
- **高度な管理機能**: 記事管理、レッスン状況管理、Instagram連携
- **保護者フレンドリー**: 分かりやすいUI/UXで情報アクセスが容易
- **リアルタイム更新**: 管理画面からの即座反映

## インストール & セットアップ

### 1. リポジトリクローン
```bash
git clone https://github.com/your-org/rbs.git
cd rbs
```

### 2. ローカルサーバー起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx serve src/public

# PHP内蔵サーバーの場合
php -S localhost:8000 -t src/public
```

### 3. アクセス
- **メインサイト**: http://localhost:8000/
- **管理画面**: http://localhost:8000/pages/admin-login.html

## 使い方

### 管理画面ログイン
- **ユーザー名**: `admin`
- **パスワード**: `rbs2024`

### レッスン状況更新（v3.0.0の新機能）

1. **日付選択**: カレンダーで対象日を選択
2. **全体ステータス設定**: 美しいカードからステータス選択
3. **コース別設定**: 必要に応じて個別調整
4. **リアルタイムプレビュー**: 「プレビュー」ボタンで表示確認
5. **保存**: 「保存して公開」で即座反映

### 新機能の活用

#### テンプレート機能
```
1. よく使う設定を作成
2. 「テンプレート保存」クリック
3. 名前を設定して保存
4. 次回以降、ワンクリックで適用
```

#### 前日コピー機能
```
1. 日付を設定
2. 「前日をコピー」クリック
3. 前日設定が自動適用
4. 必要に応じて微調整
```
